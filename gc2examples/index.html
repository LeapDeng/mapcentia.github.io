The GC2 API is a thin wrapper around Leaflet. Its makes it easy to initiate a Leaflet map with tile and vector overlays from a GC2 server. Also it defines commonly used base layers from providers like OSM, Google Maps, Bing Maps, Here Maps and Stamen.
